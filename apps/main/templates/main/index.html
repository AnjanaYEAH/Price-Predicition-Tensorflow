About

The goal of Project Avalon is to utilise existing open-source technologies in building a neural network to forecast stock prices.


Our solution applies deep reinforcement learning with TensorFlow - an open-source machine learning framework - to identify patterns in graphed historical data. Through image recognition, TensorFlow will aid us in predicting financial markets.


Financial analysts can use our software to help decide whether they should buy/sell/hold a particular stock.
